# Vinyl Audio Image

Convert music into vinyl record artwork.

This tool generates a record-style image from audio
and embeds the music data into the PNG.

You can later load the image and play the music back.

## Features

- Generate record-style artwork from audio
- Embed audio data into PNG
- Restore audio from the image
- Play / pause with rotating record
